# Bounce Back Fitness

Corrective exercise, functional training, and personalized coaching website for Bounce Back Fitness locations in Woodbridge and Metuchen, NJ.

## Local Development

```
npm install
npm run dev
```

## Build for Production

```
npm run build
```

## Deploy to Netlify

See DEPLOYMENT_GUIDE.md for step-by-step instructions.

## Tech Stack

- React 18 with TypeScript
- Vite (build tool)
- Tailwind CSS (styling)
- Radix UI / shadcn components
- Wouter (routing)

## External Integrations

- Square Booking (appointment scheduling)
- Google Forms (contact form)
- YouTube (embedded video)
- Google Analytics 4
